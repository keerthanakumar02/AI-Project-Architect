# AI Project Architect

An AI-powered web application that helps users generate software project plans, architecture documents, and downloadable PDF reports.

## Features

- AI-powered project planning
- Generate software architecture
- Download project reports as PDF
- Project history management
- Clean and user-friendly interface

## Tech Stack

- Python
- FastAPI
- HTML
- CSS
- JavaScript
- Google Gemini AI

## Project Structure

```
AI-Project-Architect/
├── backend/
├── frontend/
├── .gitignore
└── README.md
```

## How to Run

### Backend

```bash
cd backend
uvicorn app.main:app --reload
```

### Frontend

```bash
cd frontend
python -m http.server 5500
```

Open:

```
http://127.0.0.1:5500
```
